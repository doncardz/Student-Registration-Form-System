Student Registration System (Java Basic OOP Beginner Project)

This is a simple Java console-based project that demonstrates basic Object-Oriented Programming (OOP) concepts such as classes, objects, attributes, and methods.
📌 Project Description

This project allows a user to input student information such as:

- Name
- Course
- Age

After input, the system displays the student’s information using a class method.

What I Learned

Through this project, I practiced and learned:

- Creating classes and objects in Java
- Using attributes (variables inside a class)
- Creating and calling methods
- Separating input logic (Main class) from data structure (Student class)
- Basic OOP structure and design
